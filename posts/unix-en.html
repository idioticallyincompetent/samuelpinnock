Operating system

UNIX

UNIX refers to a family of operating systems developed in the 1970s that still underpins much of modern computing today — including Linux and macOS.

Design principles

UNIX is defined by a hierarchical filesystem, robust multi-user and multitasking support, and a philosophy of combining small, specialized programs through the command line.

The command line

Being comfortable with the shell — navigating the filesystem, processing text data, or automating repetitive tasks — is a foundational skill in any technical environment.

Relevance in lab and production settings

Many measurement systems, control systems, and servers run on UNIX-like operating systems, which is why solid command-line skills remain essential in technical roles.
